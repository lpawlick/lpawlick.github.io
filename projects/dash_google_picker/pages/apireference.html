    API Reference¶

    GooglePicker¶

    The Google Picker Python class

    Usage:

    GooglePicker(
    id='google-picker',
    client_id='GOOGLE_CLIENT_ID',
    developer_key='GOOGLE_DEVELOPER_KEY',
    ),

    Parameters¶

    idstr

    A unique identifier for the component

    openbool

    If the picker should be opened immediately, defaults to False

    view_idsList[str]

    The views that should be shown, essentially a filter. All possible values can be found in ViewId

    client_idstr

    The client ID for authenticating with the Google API.

    scopestr

    The api scope for this application. Defaults to https://www.googleapis.com/auth/drive.readonly

    developer_keystr

    The developer key for accessing the Google API.

    enabled_featuresList[str]

    The features that should be enabled in the picker, all values can be found in Feature. Defaults to []

    disabled_featuresList[str]

    The features that should be disabled in the picker, all values can be found in Feature. Defaults to []

    localestr

    The language of the google picker, all supported langauges can be found in the Google Picker API Documentation

    Documents¶

    Representation of the documents selected by the Google Picker

    class dash_google_picker.Documents.GoogleDocument(dict_data: Dict[str, Union[str, int, bool]])¶

    A dataclass representing a Google Document.

    The class takes a dictionary as an argument and sets the key-value pairs as attributes on the object. The dictionary should represent the properties of a Google Document as per the Google Picker API.

    Note

    This class is not intended to be manually instantiated. Instances of this class are created by the GoogleDocuments class.

    Attributes are dynamically set based on the key-value pairs in the dictionary provided.

    Parameters:

    dict_data (Dict[str, Union[str, int, bool]]) – A dictionary where the keys represent attribute names and the values represent attribute values. Should represent a Google Document as per the Google Picker API.

    Returns:

    An instance of the GoogleDocument class representing the Google Document.

    Return type:

    GoogleDocument

    class dash_google_picker.Documents.GoogleDocuments(documents_data: List[Dict[str, Union[str, int, bool]]])¶

    A class to represent a list of GoogleDocument objects.

    The class takes a list of dictionaries as an argument and creates a GoogleDocument object for each dictionary in the list.

    Usage:

    @app.callback(
    [Input('google-picker', 'documents')],
    prevent_initial_call=True
)
def display_output(documents):
    docs : List[GoogleDocument] = GoogleDocuments(documents)
    Parameters:

    documents_data (List[Dict[str, Union[str, int, bool]]]) – A list of dictionaries where each dictionary should represent the properties of a Google Document as per the Google Picker API.

    Returns:

    A list of GoogleDocument objects.

    Return type:

    List[GoogleDocument]

    Note

    Only data from the Google Picker API returned by the GooglePicker should be passed into this class.

    GooglePicker React Component¶

    The underlying react component for the Google Picker.

    Note

    This is not available in the Python Environment, only when interacting with the react component directly

    class GooglePicker(props)¶

    The constructor method is a special method for creating and initializing objects created with a class

    Arguments:

    • props (Object) – A collection of properties passed to the component

    Prop Types¶

    id¶

    A unique identifier for the component

    open¶

    Determines if the picker is opened or not

    view_ids¶

    Google View IDs to be displayed in the picker

    client_id¶

    The client_id of the Google Cloud application

    scope¶

    The scopes for the Google Cloud application

    developer_key¶

    The developer key of the Google Cloud application

    enabled_features¶

    Features to enable in the picker

    disabled_features¶

    Features to disable in the picker

    locale¶

    The locale to be used in the picker

    Default Props¶

    open¶

    The picker popup is closed by default

    view_ids¶

    No filtering by default

    scope¶

    The read only scope is enough to pick a document

    enabled_features¶

    No extra features enabled by default

    disabled_features¶

    No features disabled by default

    locale¶

    No default locale set

    Functions¶

    componentDidMount()¶

    A lifecycle method that gets called immediately after a component is mounted This method is used to load Google API scripts and initialize the Google Picker

    loadGoogleApi()¶

    This function is used to dynamically load Google API script into the HTML body

    Returns:

    Promise – Promise object represents the eventual completion or failure of loading Google API script

    loadGoogleGsiClient()¶

    This function is used to dynamically load Google One Tap script into the HTML body

    Returns:

    Promise – Promise object represents the eventual completion or failure of loading Google One Tap script

    onApiLoad()¶

    This function is called after Google API script is loaded successfully. It initializes the Google Picker

    gisLoaded()¶

    This function is called after Google One Tap script is loaded successfully. It initializes Google One Tap client

    createPicker()¶

    This function is used to create and display the Google Picker It also handles the access token needed for authorizing Google Drive access

    componentDidUpdate(prevProps)¶

    A lifecycle method that gets called when a component’s props have been changed This method is used to update the component state and create Google Picker if the open prop is changed

    Arguments:

    • prevProps (Object) – A collection of properties of the component before update

    pickerCallback(data)¶

    This function is a callback that gets called after the user selects any file/folder in Google Picker It updates the component state based on the selected files/folders

    Arguments:

    • data (Object) – The response data from Google Picker after user selection

    render()¶

    A lifecycle method that defines the render output of the component
